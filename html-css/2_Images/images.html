Supporting High-density Screens (for fixed-size images)

physical resolution -> the qualitiy of a device

logical resolution -> the size that we see on a screen
*CSS is always based on logical resolution

DPR ->the ratio of physical to logical pixels

<img
- src="images/....webp"
- alt="..."
- srcset="images/....webp 1x, images/...@2x.webp 2x"
/>

Resolution Switching-1 (flexible-sized images)

different screen sizes -> different qualitiy of image

provide 3 different sizes S, M, L
<img
- src="images/....jpg"
- alt="..."
- srcset="images/....jpg 400w, images/...@2x.jpg 800w, images/...@3x.jpg 1200w"
/>

the best way to generate different sizes of an image
