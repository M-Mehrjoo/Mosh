Absolut & Relative Position

1. by default the position of all elements is static(they are not positioned)
2. relative position means we can position the element relative to it's nomal position
3. with absolut position we can position the element relative to it's container -> we should set the position property of container to relative.

relative to the viewport fixed position means to place the element relative to the viewport
like a navigation bar that always stay on the top
