Embedding Web Fonts (using a custom fonts)

The default color for the body text is pure black (#000). It's best to change it to dark grey.

Font files come in different formats: TTF, OTF, EOT, WOFF and WOFF 2.0

We can convert any font file to a WOFF file on fontsquirrel.com.
- - Convert the fonts
- - Copy the fonts in to our Project
- - Copy the rules from font-stylesheet.css file and put them at the top of our css (update url)

Google Fonts (Font Services)

Select a font from Google Fonts.

Copy links and put them in the <head> of your html

We can edit the url any time.

System Font Stack

1. The system font stack -> the default font used by each oprating system (windows, android, mac)
2. the font looks different on different devices.
3. using system font stack make the best user exprience.

font-family: system-ui,

Sizing Fonts

html { font-size: 62.5%; } -> 62.5% of 16px = 10px

www.type-scale.com

Vertical & Horizontal Spacing

line-height: 1.5; (don't use units) -> spacing between lines

letter-spacing: -> space between letters
word-spacing: -> space between words

Ideal line length -> 50 - 70 chatacters
p { width: 50ch ; } -> 50 ch = 50 zero ~ 70 character

Formating Text

text-align: center; -> align the text Horizontaly
text-indent: -> space at the start of the line ( * p+p { } -> the spacing starts from second paragraph * )
text-decoration: -> line-through -, ...
text-transform: -> lowercase, uppercase, ...

enforce a limit on the paragraph and truncate the text (one line) ->
1- set a fixed width
2- white-space: nowrap; -> for controlling wrapping;
3- overflow: hidden;
4- text-overflow: ellipsis;

column-count: -> creating multi colum text
