Creating a Basic Form

<form action=" ">
- <label for="...">...</label>
- <input id="..." type="text" />
</form>

label element -> the name of the data that comes in input box

input element -> to accept data from the user

type attribute -> determines the type of the input and can be:
- text | - password | - email | - checkbox | - radio | - range | - date | - file | etc.

Styling Forms

put input below its lable -> lable {display: block}

to avoid applying the styles to all the input types -> use attribute selector -> input[type="..."] { }

to style our input in focus state -> use psudo class selector -> input[type="..."]:focus { }

* outline: 0; -> to remove the border that be applyed by default around input-box or buttons

* transition: border-color 0.2s, box-shadow 0.2s; -> not disappear quickly
